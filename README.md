## Installation / インストール

### Prerequisites / 前提条件
- Rust 1.70+ (Install via [rustup](https://rustup.rs/))

### Build & Run / ビルド・実行

```bash
# Clone repository / リポジトリをクローン
git clone https://github.com/BonoJovi/Baconian.git
cd Baconian

# Build / ビルド
cargo build

# Run / 実行
cargo run
```
